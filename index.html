What is it?

S3 Slider is a simple slideshow that displays images with their description.
This is an complete rewrite to make it smaller / simpler / more elegant.

It is simple to use, very small(~1kb for js+css) and elegant! Demo(older version, new looks similar) or Live demo

Original license is for free private/commercial use.

Download

Requires jQuery >= 1.3.

You need s3-slider.js and s3-slider.css

Works in IE6 + the rest.

Demo

Simple images with some misc text.

At the start IE6 will see an instant fade from blank->image0, if you do not like this make the first slide visible(display:block).

Naturally the timeout should never be smaller then 2 * fadeTime...
